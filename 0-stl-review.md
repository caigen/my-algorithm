### priority_queue as Heap


### unordered_map as Hash

```c++
std::unordered_map<int, int> hash;
for (auto iter = hash.begin(); iter != hash.end(); ++iter)
{
	std::cout << iter->first << "," << iter->second << std::endl;
}
```



### sort

